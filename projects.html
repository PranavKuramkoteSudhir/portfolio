Toxic Comment Analyzer

Built a Natural Language Processing (NLP) model to detect and classify toxic comments in social media discussions. Preprocessed text data using TF-IDF, word embeddings, and tokenization techniques to improve model accuracy. Trained multiple models including Logistic Regression, Random Forest, and LSTM, achieving high precision and recall scores. Developed a Flask-based API to integrate the model into web applications for real-time comment analysis. Deployed on AWS Lambda for cost-efficient, scalable inference.

Employee Attrition Prediction

Developed a predictive analytics model to identify key factors influencing employee attrition. Used Pandas, Seaborn, and Matplotlib for exploratory data analysis (EDA) to uncover trends in HR datasets. Implemented Random Forest, Gradient Boosting, and Logistic Regression models, achieving 85% accuracy. Deployed a Flask web app that allows HR professionals to input employee details and get attrition risk predictions. Integrated model deployment on AWS EC2, ensuring a scalable cloud-based solution.

Ensured scalability using S3 for storage and API Gateway for secure access.
