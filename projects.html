Toxic Comment Analyzer

Built a Natural Language Processing (NLP) model to detect and classify toxic comments in social media discussions. Preprocessed text data using TF-IDF, word embeddings, and tokenization techniques to improve model accuracy. Trained multiple models including Logistic Regression, Random Forest, and LSTM, achieving high precision and recall scores. Developed a Flask-based API to integrate the model into web applications for real-time comment analysis. Deployed on AWS Lambda for cost-efficient, scalable inference.

Mail classifier

I built a custom GPT-style transformer from scratch using only NumPy and core Python. It includes multi-head attention, layer normalization, and greedy decoding logic. I trained it on email data and evaluated the results with custom metrics. The model can generate replies and summaries for real emails. This project deepened my understanding of LLMs beyond library abstraction.

Rag based chatbot to answer recruiter questions

I built a serverless AI-powered chatbot using React, Node.js, and Tailwind CSS. It leverages Cohere's APIs for both semantic search (embeddings) and natural language generation. PostgreSQL with pgvector on Render is used to store and retrieve relevant context. The backend is deployed as serverless functions via Vercel. This chatbot can answer questions about my experience, skills, and projects in real-time.
